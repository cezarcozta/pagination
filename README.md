### Getting Started  

> **Yarn and NPM REQUIRED**  

```bash
# Clone this repository
$ git clone https://github.com/cezarcozta/pagination.git  

# Access project folder
$ cd pagination

# Install all dependencies
$ yarn

# Run the project
$ yarn start

# Localhost
http://localhost:3000
```
